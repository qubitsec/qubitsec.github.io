Datosはウェブ上のトラフィックを監視するためのツールです。 監視のためにインターフェイスのポート情報のみを使用するため、他のプロセスとの衝突は発生しません。

次のような環境で使用できます。

- ウェブサーバーでApache HTTPD、Nginx ではない環境で例えば、Apache Tomcat単独使用の場合

- コンテナ環境で、ホストに複数のウェブPODを使用する場合
