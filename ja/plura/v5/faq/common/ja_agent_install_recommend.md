## PLURAをインストールするための最小/推奨仕様は下記の通りです。

<br />

### 1. Windows

     最小仕様 : CPU Intel Pentium4 1GHz / Memory 2 GB以上
     推奨仕様 : CPU Intel Core i5 2GHz / Memory 4 GB以上

<br />

### 2. Linux

     最小仕様 : CPU Intel Pentium4 1GHz / Memory 1 GB以上
     推奨仕様 : CPU Intel Core i5 2GHz / Memory 2 GB以上
